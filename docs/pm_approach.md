I will use a **hybrid approach between Scrum and Kanban** to manage the project. Scrum elements (short weekly sprints, clear goals, and regular reflection) will help maintain structure and track progress, while Kanban will be used for day-to-day task management through a visual board (e.g., To Do → In Progress → Done). This allows flexibility during development, which is important for an experimental project involving AI features, while still ensuring consistent progress and accountability.

Responsibilities will be divided between me and AI tools to maximize efficiency. I will focus on **architecture design, key technical decisions, and code quality control**, including reviewing and refining all generated code. AI will assist with **boilerplate code generation, unit tests, documentation drafts, and quick implementation suggestions**. However, all AI-generated output will be critically evaluated, tested, and adapted to fit the project’s architecture and requirements, ensuring that the final system remains consistent, maintainable, and well-understood.
